# 🚀 NEWSLENS.AI

## 💡 Overview

* AI-based smart news analysis platform
* Summarizes, analyzes, and converts news into engaging formats

## 🔥 Key Features

* Fake news detection (AI-based probability)
* Bias detection (Left / Right / Neutral)
* Short news summarization (NLP)
* Text-to-video generation (AI avatar + voice)
* Multi-language translation

## 🧠 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* AI: OpenAI / HuggingFace
* ML: Classification models
* TTS: gTTS / ElevenLabs

## 🎯 Goal

* Detect fake & biased news
* Save time with quick summaries
* Provide modern news formats (video/audio)

## 🚀 Future Scope

* Real-time news scraping
* Personalized feed
* Mobile app
* Social media integration

## 💡 Unique Point

* Combines NLP + ML + AI video generation in one platform

👉 Smart AI-powered news assistant
